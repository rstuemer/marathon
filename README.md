# marathon
Tool to Manage Simple Running Events 

## Vision 
The Goals is to have a simple webbase app to manage running event ..... 

### Implementation Details/ Goals 
   + Codeingiter
   + MariaDb
   + Docker/Podman Container
   + Simple Login and Backend for Maintainer
   + Stats Display for Runners on Smartphone or public display 
   + Simple Time Measurement with manual startnumber input (later add modern measuerment with RFID or other stuff)
   + Multiple Competions per Event (Fullmarathon, Half marathon ...)
   + Multiple Startinggroups per Competions
   + Subclasses per Competions (Age, gender ... )
